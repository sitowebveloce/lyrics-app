# Find Lyrics App

Simple app to find music lyrics.
![](img/lyrics.gif)
Type the name of the artist or the song name inside the input field.

## No framework
Realized using basic HTML CSS and Vanilla Javascript.


## Usage
```
Open 'index.html'.
```
